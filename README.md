# Trig Explorer

An interactive visualization of sine, cosine, and tangent built to help students understand trigonometry intuitively.

## Features

- **Interactive circle** — drag the point around the unit circle (or any radius) to change the angle
- **Color-coded triangle** — the hypotenuse, adjacent side (cos), and opposite side (sin) are drawn and labeled with their actual lengths
- **Live trig values** — sin, cos, and tan update in real time with animated bar graphs
- **sin and cos function graphs** — shown below the circle with the current angle marked, so you can see where you are on the full wave
- **Degrees or radians** — toggle between units; radians are displayed as clean π fractions (π/4, 3π/2, etc.) where possible
- **Adjustable radius** — start with the unit circle (r = 1) or explore how scaling affects the side lengths while the ratios stay the same

## How to use

No installation needed. Just open `index.html` in any modern web browser.

**Controls:**
- Drag the purple dot on the circle to change the angle
- Use the **Angle** slider or type directly into the angle box (accepts `45`, `45°`, `pi/4`, `π/4`, `3pi/2`, `1.5708 rad`, etc.)
- Use the **Radius** slider to change the circle size
- Toggle between **Degrees** and **Radians** at any time
